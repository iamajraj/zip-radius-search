Zip Radius Search using Zip Code (USA)

API -> /api/search?zip={zipcode}&distance={e.g 5}&type={mile/km}

Note:-

Both zip, distance and type queiries are required!!

Usage:-

zip: {zip code to search with}

distance: {radius distance in mile or km}

type: {mile or km according to your provided distance value}

Sample example

Try the below query->

/api/search?zip=14466&distance=25&type=km

Response:-

status: {either 'success' or 'failed'}

results: [{ zipcodes }]
